# 快速开始指南

## 5分钟上手物流路径优化系统

### 第一步：准备数据文件

#### 方法A：使用示例数据（最快）

项目已提供示例数据，直接使用：

```bash
python scripts/logistics-router-interactive.py
```

选择 `2` 从CSV文件导入，输入 `example_data`

#### 方法B：创建自己的数据

创建一个新目录，例如 `my_city_data/`，包含3个CSV文件：

**nodes.csv**（城市节点）
```csv
id,latitude,longitude,type
仓库A,39.9042,116.4074,warehouse
客户1,39.9100,116.4200,delivery
客户2,39.8950,116.3900,delivery
```

**edges.csv**（道路）
```csv
from_node,to_node,distance,min_time,max_time
仓库A,客户1,5.2,10,20
仓库A,客户2,7.8,15,30
客户1,客户2,3.5,7,14
```

**deliveries.csv**（配送任务）
```csv
id,demand,service_time,priority,time_window_start,time_window_end
客户1,50,15,high,2024-01-15 09:00:00,2024-01-15 12:00:00
客户2,30,10,medium,2024-01-15 10:00:00,2024-01-15 14:00:00
```

## ❓ 常见问题

**Q: 我的数据格式对吗？**
A: 运行程序选择选项4导出模板，参照模板填写

**Q: 导入失败怎么办？**
A: 检查：1) 文件在正确目录 2) CSV格式正确 3) 编码为UTF-8

**Q: 能处理多少个配送点？**
A: 最近邻算法可处理1000+点，遗传算法推荐100以内，动态规划推荐10以内

**Q: 结果不满意怎么办？**
A: 1) 尝试其他算法 2) 调整算法参数 3) 增加车辆数量
